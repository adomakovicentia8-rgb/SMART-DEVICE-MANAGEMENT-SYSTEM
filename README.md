#OOP MINI PROJECT

## Description
This project is an Object-Oriented Programming mini project developed in Python. It simulates the management of smart devices in a smart home.

## Features
- Parent class (SmartDevice)
- Inheritance
- Encapsulation using private attributes
- Getters and setters
- Constructors (__init__)
- super() method
- Menu-driven interface
- Smart Light
- Security Camera
- Temperature Sensor

## How to Run
1. Install Python 3.
2. Save the program as `smart_device_management.py`.
3. Open a terminal.
4. Run:

python smart_device_management.py

5. Use the menu to interact with the devices.
